Лабораторная работа №1: Последовательное перемножение матриц

Цель работы
Реализовать программу на C++ для перемножения двух квадратных матриц и провести исследование времени выполнения в зависимости от размера матриц.

Результаты экспериментов

Размер матрицы | Время (с)
200 | 0.082
400 | 0.387
800 | 1.566
1200 | 6.428
1600 | 16.437
2000 | 45.268

Верификация
Все результаты проверены с помощью библиотеки numpy. Результаты совпадают с эталоном.

Анализ
При увеличении размера матрицы в 2 раза время выполнения увеличивается примерно в 8 раз, что соответствует теоретической сложности O(N³).

Запуск программы
lab1.exe 200 - для матрицы 200x200
lab1.exe 400 - для матрицы 400x400
